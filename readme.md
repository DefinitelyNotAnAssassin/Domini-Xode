# Domini Xode 

A Single Page Application created using Django, HTMX, Markdown, OpenAI API, and etc. 

Currently hosted at: [tdc.pythonanywhere.com](tdc.pythonanywhere.com)


# Functionalities 
1. Markdown Supported Articles
2. Dynamic Content Generation
3. Dynamic Event Status
4. Contact Us Saves Locally and Email
5. Tools
   - Coding AI
   - HTML Prettier
   - Speech to Text Translation
6. Partitioned HTML Files (faster loading for subsequent pages)
